О приложении

BnGarage — приложение для учёта автомобилей, истории технического обслуживания и умных напоминаний. Все данные хранятся только на вашем устройстве — без регистрации и без облака.

Возможности

  • Добавление и управление несколькими автомобилями
  • Полная история сервисного обслуживания с фото
  • Умные напоминания по дате и пробегу
  • Учёт расходов на каждое авто
  • Работает полностью офлайн
  • Интерфейс на русском, English и O'zbek tili

Часто задаваемые вопросы

Где хранятся мои данные?

Только на вашем устройстве. Никакие данные не передаются на сервер — ни автомобили, ни история ТО, ни фотографии.

Нужен ли интернет для работы?

Нет. Приложение работает полностью офлайн. Интернет не требуется ни для одной функции.

Как настроить напоминания?

При добавлении или редактировании записи об обслуживании укажите следующую дату или пробег — приложение создаст уведомления автоматически (за 30 дней, 7 дней и в сам день).

Можно ли восстановить данные после переустановки?

Данные хранятся локально. Рекомендуем делать резервную копию через iTunes / Finder (iOS) или Google Drive (Android).

Приложение платное?

Нет. BnGarage полностью бесплатен — без подписки, рекламы и скрытых платежей.
