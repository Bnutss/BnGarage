1. Общие положения

Настоящая Политика конфиденциальности описывает, как приложение BnGarage обрабатывает информацию пользователей. Мы уважаем вашу конфиденциальность и стремимся защитить ваши личные данные.

2. Сбор данных

BnGarage не собирает и не передаёт персональные данные на внешние серверы. Все данные, которые вы вводите в приложение (автомобили, записи об обслуживании, напоминания), хранятся исключительно на вашем устройстве в локальной базе данных.

3. Разрешения приложения

  • Камера — используется только для фотографирования автомобиля. Фото хранятся на вашем устройстве.
  • Галерея (Фото) — используется для выбора изображения из библиотеки фотографий. Доступ к другим файлам не запрашивается.
  • Уведомления — используются для отправки напоминаний о ТО и страховке. Уведомления формируются локально, без подключения к интернету.

4. Передача данных третьим лицам

Мы не продаём, не обмениваем и не передаём ваши данные третьим лицам. Приложение не требует регистрации и не имеет учётных записей пользователей.

5. Аналитика

Приложение использует Firebase для базовой диагностики сбоев (crash reporting). Эти данные анонимны и не содержат личной информации. Они используются исключительно для улучшения стабильности приложения.

6. Хранение данных

Все пользовательские данные хранятся локально на вашем устройстве. При удалении приложения все данные удаляются вместе с ним. Мы рекомендуем регулярно создавать резервные копии через стандартные средства iOS (iCloud/iTunes) или Android (Google Drive).

7. Дети

Приложение не предназначено для детей младше 4 лет и не собирает данные о детях намеренно.

8. Изменения политики

Мы оставляем за собой право обновлять данную политику. Об изменениях мы будем уведомлять через обновление приложения. Дата последнего изменения указана в заголовке страницы.

9. Контакты
